# Meetings

One file per meeting. Filename: `YYYY-MM-DD-slug.md`.

Granola transcripts dump here. Sales call notes go here. Then either:
- Distilled into a project file, or
- Linked from a person file

Naming examples:
- `2026-04-29-vincenzo-tax-coach.md`
- `2026-05-06-junta-comunale-adcab.md`
- `2026-05-12-ddp-partners-demo.md`
