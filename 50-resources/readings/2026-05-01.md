### **Claude Cowork + Seedance 2.0** (Mike Futia)
*Apr 27, 2026*

**New workflow:** Automated UGC (user-generated content) video ad production using Claude Cowork + Seedance 2.0 + Actor generation tools.

**The problem UGC solves:** Testimonial-style videos from "real people" (not obvious actors) convert 2-3x better than polished ads. But hiring real UGC creators = $200-500 per video, 1-2 week turnaround, inconsistent quality.

**Seedance 2.0 improvements:**
- Better lip-sync (fixed the "uncanny valley" mouth movements from v1)
- Consistent character memory (same person across multiple videos)
- Emotion control (happy, concerned, excited, skeptical)
- Background customization (kitchen, office, car, bedroom)

**Full workflow:**
1. *Character creation* - Generate 5-10 diverse "UGC creator" characters with Higgsfield or similar tool, save to character library
2. *Script writing* - Claude Cowork writes 10 video scripts following proven UGC frameworks (problem-agitation-solution, before-after-bridge, etc.)
3. *Video generation* - Seedance batch-generates all videos using saved characters + scripts
4. *Post-production* - CapCut automation adds captions, background music, subtle zoom effects
5. *Meta upload* - Claude uploads directly to Meta Ads via new Ads CLI

**Time investment:** ~2 hours setup (character creation), ~30 minutes per batch of 10 videos thereafter.

**Cost:** ~$0.50-2 per video (vs. $200-500 for real UGC creators)

**Quality vs. real UGC:** Not quite as good as best real creators, but better than 80% of Fiverr UGC. Biggest tell: sometimes background details look slightly off under scrutiny.

**Best use case:** High-volume testing (generate 50 variations, let Meta algorithm find winners) rather than hero creative.

**Claude Cowork Artifact included:** Full production pipeline from product description input to Meta-ready video files. Generates character prompts, scripts, Seedance API calls, CapCut automation commands, Meta upload commands.
